# Tech Stack
<br/>

The list of the most relevant libraries/frameworks used in this project:

* [React](https://pl.reactjs.org/) - A JavaScript library for building user interfaces.
* [Next.js](https://nextjs.org) - The React Framework with hybrid static and server rendering option.
* [React Query](https://react-query.tanstack.com/) - data-fetching library for React.
* [styled-components](https://styled-components.com/) - CSS in JS for styling React component systems.
* [React Testing Library](https://testing-library.com/docs/react-testing-library/intro/) - a very light-weight solution for testing React components. 
* [MobX](https://mobx.js.org/README.html) - simple, scalable state management.
* [React hook form](https://react-hook-form.com/) - performant, flexible and extensible forms with easy-to-use validation.
* [next-i18next](https://github.com/isaachinman/next-i18next) - management of translation content.

Before contributing to the project, we recomend **reading the documentations** for each of the module.
